# 🏭 Caso 5: Control Industrial - Filtro Notch

## 📋 Descripción
Diseño e implementación de un filtro Notch (rechazo de banda) Butterworth para eliminar resonancia en turbinas industriales.

## 🎯 Objetivo
Eliminar la resonancia de 120 Hz que enmascara vibraciones anómalas, permitiendo mantenimiento predictivo preciso.

## 📊 Resultados Clave
| Métrica | Valor |
|---------|-------|
| Atenuación Resonancia | +22.8 dB |
| Reducción Anomalías | 83.3% |
| Energía Preservada | 92.4% |
| Mejora SNR | +15.6 dB |

## 🛠️ Tecnologías
- Python 3.10+
- SciPy (filtrado Notch)
- Matplotlib (visualización)
- HTML5/CSS3/JS (interfaz web)
