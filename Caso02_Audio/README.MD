# 🎵 Caso 2: Restauración de Audio Histórico

## 📋 Descripción
Restauración de grabaciones históricas de discursos políticos (1950) mediante filtro digital pasa altos Chebyshev para eliminar zumbido de 60 Hz.

## 🎯 Objetivo
Eliminar el zumbido de fondo de 60 Hz y ruido de grabación analógica preservando la inteligibilidad del discurso.

## 📊 Resultados Clave
| Métrica | Valor |
|---------|-------|
| Mejora de SNR | +25.8 dB |
| Reducción THD | 67.4% |
| Reducción Zumbido | 42.3 dB |
| Inteligibilidad | 89.5% |

## 🛠️ Tecnologías
- Python 3.10+
- SciPy (filtrado Chebyshev)
- Matplotlib (visualización)
- HTML5/CSS3/JS (interfaz web)
