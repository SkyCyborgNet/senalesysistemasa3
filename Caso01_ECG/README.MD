# 🫀 Caso 1: Monitoreo de ECG en UCI

## 📋 Descripción
Diseño e implementación de un filtro digital pasa bajos Butterworth para eliminar ruido en señales electrocardiográficas (ECG) en entornos de cuidados intensivos.

## 🎯 Objetivo
Mejorar la calidad de la señal ECG para facilitar la detección temprana de arritmias y reducir falsas alarmas en la UCI.

## 📊 Resultados Clave
| Métrica | Valor |
|---------|-------|
| Mejora de SNR | +33.6 dB |
| Correlación | 0.97 |
| Energía preservada | 94.2% |
| Frecuencia cardíaca | 72.3 BPM |

## 🛠️ Tecnologías
- Python 3.10+
- SciPy (filtrado)
- Matplotlib (visualización)
- HTML5/CSS3/JS (interfaz web)
